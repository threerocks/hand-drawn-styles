# MJ 标杆对照实验(2026-07-13)

目的:gpt-image(codex)出图"数字感偏重",以 Midjourney V7 为材质标杆,迭代 prompt 直到 codex 接近。产出 = STYLES.md 附录 A「实拍纸质感增强层」。

- 统一场景见 `scene.md`;四画风:3.1 蜡笔童涂潦草自画 / 8 蜡笔实拍 / 1.1 儿童涂色低饱和 / 11 情绪叙事淡彩速写。
- `mj/sXX-*.png` = MJ V7 标杆(四宫格挑格,grid-*.png 为原格);`codex-r1/` = 仓库原配方基线;`codex-r2/` = 叠加材质增强层;`compare/` = 三列对照图。

## 结论

| 画风 | r1 基线主要差距 | r2 叠层后 |
|---|---|---|
| 3.1 | 干净数字蜡笔、纸面过平 | 纸纹/运笔方向/蜡质/折痕齐备,≈MJ,场景更准 |
| 8 | 纸不够"实拍",填色偏满 | 皱纸摄影感到位,稀疏笔触+压力变化,≈MJ |
| 1.1 | 线稿过稳、填色过匀、纸面平 | 马克笔渗墨+蜡笔逃线+实拍纸,≈MJ |
| 11 | 本就接近,仅纸面偏平 | 纸牙口+颜料沉淀,持平或优于 MJ(且构图更守纪律) |

附注:MJ 全程无视"低饱和/留白"类纪律约束(审美先验覆盖指令),标杆仅取其材质;叙事与配色纪律仍是 gpt-image 强项——两者结合的方式就是附录 A。
