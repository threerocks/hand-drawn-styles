# 手绘风格 prompt 生成器

本 skill 把用户要画的内容,套进一种内置画风的提示词配方,产出可直接复制去喂图像模型的**最终 prompt 纯文本**。它本身**不生图**。

## 执行流程

### 1. 确定画风

- **用户已显式指定** → 直接采用。识别方式三选一(任一命中即可):
  - 中文名:纯人类手绘 / 儿童涂色 / 极简线条 / 黑白漫画 / 蜡笔 / 童涂 / 吉卜力 / 小豆人 / 涂鸦
  - 编号:1–5
  - 英文别名:`handmade` `childlike-coloring` / `minimal-line` `ink-sketch` / `crayon` `kid-crayon` / `ghibli` / `bean` `doodle`
- **用户未指定** → 展示下面这个菜单,**停下等用户选**,不要自己替他挑:

  ```
  请选择画风(回复编号或名字):
  1. 纯人类手绘儿童涂色页 —— 大人画粗黑线稿+孩子填色,真实纸面拍摄感,适合讲故事/亲子
  2. 极简黑白线条漫画手稿 —— 粗糙速写、不闭合线条、高对比留白,独立漫画感
  3. 蜡笔童涂 —— 5岁小孩用蜡笔画的笨拙"坏画",歪扭出框、引人发笑
  4. 吉卜力风 —— 柔和水彩、暖光、治愈梦幻的手绘动画感
  5. 小豆人涂鸦信息图 —— 黑色圆豆人讲解图,单橙点缀、手绘箭头标注,竖版多格
  ```

### 2. 取配方

读 [STYLES.md](STYLES.md),取对应编号那段完整提示词模板。

### 3. 自动填充占位符

模板里的 `【主体】【标题词】【主色调】【N】【分镜列表】` 等占位符,**从用户给的内容自动推断填好**,不要把括号或占位符原样留在输出里:

- `【主体】` `【标题词】`:从用户描述里提取核心物体/场景词。
- `【主色调】`:按主体合理选一个主色。
- `【N】` `【分镜列表】`(仅风格5):根据用户内容拆成合理的几格步骤/要点,自己定 N 和每格内容;若内容拆不出多步,默认 3–4 格。
- 拿不准也优先合理推断,不要为占位符反复追问用户。

### 4. 处理比例(关键:不硬锁)

- 用户**传了**比例(如 16:9 / 竖版 / 方图)→ 用用户的。
- 用户**没传**:
  - 纯风格(1、3、4)→ **不注入任何比例**。
  - 版式风格(2、5)→ 只注入**软结构提示**(如"竖版、多格自上而下堆叠"),**不写具体数字比例**。

### 5. 输出

只输出最终 prompt,放进代码块方便整段复制。**不要生图,不要附加解说**(除非用户另外问)。结尾可以用一行说明用的是哪种画风 + 是否注入了比例提示。
